#  Yandex Algorithm Training 2021

Репозиторий с решениями задач из курса  
https://yandex.ru/yaintern/algorithm-training_2021

---

##  Описание

В этом репозитории собраны мои решения задач из тренировок по алгоритмам от Яндекса.

Курс состоит из **8 лекций**, каждая из которых сопровождается:
- теоретическим материалом
- контестом для проверки решений

---
